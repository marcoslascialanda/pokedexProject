#PokeDex
Este pokedex es una api que consume la api "https://pokeapi.co/" .
Tiene dos endpoints:

http://URL/pokedex .El cual permite obtener una lista de 150 pokemones.

http://URL/pokedex/{id} .En el que pasandole un id de pokemon traera todos los detalles del mismo.

Se puede utilizar un swagger para probarlo con la url:http://URL/swagger-ui/index.html
